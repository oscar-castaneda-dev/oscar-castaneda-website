Hello world. Writing a blog again takes me back to that distant 2008, when I had a blog hosted on Blogger, hoping to share my thoughts online.

17 years have passed since then, and although it may seem like I'm going in the opposite direction, where the focus is on videos, I'm excited about the idea of writing again, but this time documenting my work as a software developer.

I wanted to start this project from scratch, programming everything myself with the intention of showing my work and documenting it, but this time from a software development perspective.

This project is open source and can be found here. For now, it has a very basic stack, as I wanted to deploy something as quickly as possible, using Markdown files for the blog structure. However, in the coming weeks, I'll be implementing a backend to replace this.
